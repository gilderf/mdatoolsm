mdadata

mdadata is a class for storing, manipulation and visualisation of two-way datasets.

   data = mdadata(values);
   data = mdadata(values, rowNames, colNames);
   data = mdadata(values, rowNames, colNames, dimNames, name);

The mdadata class extends possibilities given by matrices with ability to use row and column names, exlude columns and rows without removing them physically, work with qualitative data, make interactive plots and many others.

The mdadata objects can be manipulated in a similar way as matrices, including subsetting, arithmetic and logical operations as well as some mathematical functions can be applied directly to the objects. The result of any operation is also an mdadata object. Look at the Getting started guide for more details, with a lot of examples.

Properties (main)

valuesumeric matrix with data values
rowNamesell array with unique row names
colNamesell array with unique column names
dimNamesell array with dimension names (columns and rows)
nameame of the dataset (short text string)
infonformation about the dataset (text string)

Properties (extra)

numValuesumeric matrix with data values without factors
rowFullNames ell array with full text row names
colFullNamesell array with full text column names

Examples

   a = [175 80; 181 78; 165 55];
   d = mdadata(a);
   show(d);

   % make a dataset with names for rows, columns and dimensions
   d = mdadata(a, {'Bob', 'Eva', 'Ron'}, {'Height', 'Weight'}, {'People', 'Parameters'});
   show(d);

   % add a name and information for the dataset
   d.name = 'People';
   d.info = 'People dataset from 2014';
   show(d);

 printrints information about the data and the data values

+includerowsnclude (unhide) rows

+showexcludedrowshows names and numbers of excluded rows
+removecolsemove columns from dataset
+removerowsemove rows from dataset

methods for manipuation and analysis of qualitative data.

Methods for making grouped plots

some of the standard plots can also be made for groups of objects or observations. In this case the groups will be shown using different colors or plot properties (marker, line style) and legend can added.

+gscatterUI version of scatter plot
+gplotUI version of line plot
+gbarUI version for bar plot

Methods for interactive graphics

most of the plotting methods have also GUI version, which allows to analyse data interactively, e.g. switch columns, look at statistics, exclude objects and variables, etc.

+guiplotUI tool for most of the plots
