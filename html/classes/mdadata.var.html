mdadata.var

var calculates variance for each column of dataset, or for one column, which values are split into groups.

   s2 = var(data);
   s2 = var(data, factors);

By default, method calculates variance for each column of the data object. However if a dataset with factors is provided the method will calculate the variance only for the first column of data by splitting its values into groups, corresponded to all possible combination of the factors.

Examples

   load people

   % calculate variance for all columns
   s2 = var(people);
   show(s2)

   % mark columns "Sex" and "Region" as a factors
   people.factor('Sex', {'Male', 'Female'})
   people.factor('Region', {'A', 'B'})

   % calculate variance for "Height" separately for each sex and region.
   s2 = var(people(:, 'Height'), people(:, {'Sex', 'Region'}));
   show(s2)
