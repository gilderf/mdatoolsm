mdadata.sort

sort sort rows of dataset

   sort(data, columns);
   sort(data, columns, mode);

The method sorts rows according to its values in the specified columns. The argument columns can be a vector of column numbers or a cell array of column names. Argument mode can have values ascend (default) or descend.

Examples

   load people

   people.sort('Height')
   disp(people)

   people.sort({'Sex', 'Region'}, 'descend');
   disp(people)
