mdadata.median

median calculates median value for each column of dataset, or for the first column, which values are split into groups.

   md = median(data);
   md = median(data, factors);

By default, method calculates median for each column of the data object. However if a dataset with factors is also provided the method will calculate the value only for the first column of data by splitting its values into groups, corresponded to all possible combination of the factors.

Examples

   load people

   % calculate median for each column
   md = median(people);
   show(md)

   % mark columns "Sex" and "Region" as a factors
   people.factor('Sex', {'Male', 'Female'})
   people.factor('Region', {'A', 'B'})

   % calculate median for "Height" separately for each sex and region.
   md = median(people(:, 'Height'), people(:, {'Sex', 'Region'}));
   show(md)
